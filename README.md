# Movie Series Catalog

ASP.NET Core MVC application for browsing catalog entries, managing actors and directors, and collecting user reviews.

## Review behavior

- Signed-in users can submit one review per catalog item.
- Once a review is submitted, the reviews page hides the add action for that user.
